# Auracle

AuraCle is a cutting-edge Android app created with the Kotlin programming language that is aimed
to deliver an enhanced podcast listening experience. It uses the ListenNotes API to allow users to log
in, stream podcasts, mark favorites, and record listening history. AuraCleâ€™s goal is to create a seamless and personalized user experience, hence setting a
new standard for podcast apps.

You can have a tour to our [app](https://youtu.be/WgvQlZM54JA?si=KyPrcMpLnrWDXqMf)

## How to run the App

- Run [Andriod Studio](https://developer.android.com/studio?gclid=Cj0KCQjwk5ibBhDqARIsACzmgLQ6-Umb73e9AryWkCu2lnhz871WmSVvMFUHV6cySQQC7muLY6OSbCQaApgQEALw_wcB&gclsrc=aw.ds)
- Clone the [Repository](https://github.com/anika235/AuraCle.git)
- SDK Loction needs to be changed for different setup.
- Connect to your emulator or phone
- Listen to your favorites podcasts
